PROGRAMAS UTILIZADOS EN EL PROYECTO

1. Visual Studio Code
   - Versión: VSCodeUserSetup-x64
   - Uso: Editor de código para Python y SQL.
   - URL de descarga: https://code.visualstudio.com/

2. Python 3.14
   - Arquitectura: AMD64 (64-bit)
   - Uso: Lenguaje de programación para los scripts de conexión y manipulación de la base de datos.
   - URL de descarga: https://www.python.org/downloads/release/python-3140/

3. Oracle Database Express Edition 11g (Oracle XE 11.2.0.2.0)
   - Plataforma: Windows 64-bit
   - Uso: Base de datos para almacenar tablas y registros del proyecto.
   - URL de descarga: https://www.oracle.com/database/technologies/xe-downloads.html

4. Oracle Instant Client 23.9
   - Versión: instantclient-basic-windows.x64-23.9.0.25.07
   - Uso: Permite la conexión desde Python a Oracle sin necesidad de tener instalado todo Oracle Client.
   - URL de descarga: https://www.oracle.com/es/database/technologies/instant-client/winx64-64-downloads.html

Notas:
- Todos los programas fueron instalados en un entorno Windows 10/11.
- Python y el Instant Client se configuraron en la variable PATH para que los scripts puedan ejecutarse correctamente.
- Visual Studio Code se utilizó únicamente como IDE para desarrollo de scripts y la escritura de los archivos .py
